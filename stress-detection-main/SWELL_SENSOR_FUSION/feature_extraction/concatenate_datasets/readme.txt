Folder`s contents:
	data_time: Χρονικά χαρακτηριστικά για κάθε καταγραφή
	data_freq: Συχνοτικά χαρακτηριστικά για κάθε καταγραφή
	facereader_data: Facereader χαρακτηριστικά για κάθε καταγραφή
	kin_data: Kinect Data χαρακτηριστικά για κάθε καταγραφή
	time_domain_signals_new_time: Υποδειγματοληπτιμένα physio σήματα (για κάθε 4 δείγματα του αρχικού σήματος κρατάω 1, ρίχνω
								  την συχνότητα δειγματοληψίας από τα 2048Hz στα 512 Hz.
	time_features_new_time: Χρονικά χαρακτηριστικά για κάθε καταγραφή με τα νέα σήματα.
	freq_features_new_time: Συχνοτικά χαρακτηριστικά για κάθε καταγραφή με τα νέα σήματα.

check_data.py: Συνένωση καταγραφών για κάθε συμμετέχοντα και συνδυασμός για εξαγωγή εννιαίων
			   dataset ανά αισθητήρα. 
giant_merger.py: Συνένωση των εννιαίων datasets σε ένα ολικό dataset. Γίνεται με βάση το participant_id,
				 session_num, timestamp και condition.
hp_tuning.py: Εκπαίδευση των αλγορίθμων με αυτά τα dataset και εξαγωγή υπερπαραμέτρων και μετρικών.
sk_data.py: Απλή εκπαίδευση χωρίς εύρεση υπερπαραμέτρων (χρησιμοποιούνται οι ήδη γνωστές από τον παραπάνω αλγόριθμο) και
			εξαγωγή μετρικών.
cfs_attempt.py: CFS εφαρμογή.
